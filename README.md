# FirstMyGem

https://rubygems.org/gems/first_my_gem

## Installation

```bash
bundle add first_my_gem
```

If bundler is not being used to manage dependencies, install the gem by executing:

```bash
gem install first_my_gem
```

## Usage

```rb
puts FirstMyGem.hello("your_name")
# => Hello, your_name!
```
